# cybersecurity-portfolio
Cybersecurity portfolio demonstrating vulnerability management, network security , cloud security, and OT/ICS security through hands-on projects including network scanning, penetration testing,  and security analysis.

## Tools Used

- Autopsy
- CyberChef
- Git
- GitHub
- Kali Linux
- Metasploitable 2
- Nmap
- VirtualBox
- Wireshark

## Projects

### Kali Linux – Metasploitable Penetration Testing Lab
Performed reconnaissance and service enumeration against a vulnerable Metasploitable 2 machine using Kali Linux.

Techniques used:
- SYN scanning (nmap -sS)
- Service version detection (nmap -sV)
- Aggressive scanning (nmap -A)
- SSH connectivity testing
- Enumeration of open services

Project folder: kali-metasploitable-penetration-test


### XP Cyber Range – Dangerous USB Drive Forensics Challenge
Performed forensic analysis on a suspicious USB drive image in the XP Cyber Range lab to identify malicious artifacts and investigate potential compromise.

Techniques used:
- Log analysis
- File artifact investigation
- Data extraction
- Incident analysis

Project folder: xp-cyber-range-challenges
